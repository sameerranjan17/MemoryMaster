Memory Master

Memory Master is a high-performance, web-based sequence memory game inspired by the classic "Simon Says." It challenges players to observe, remember, and replicate an ever-increasing string of random colors and sounds.

🧠 Technical Overview

This project was built to master the core fundamentals of the JavaScript DOM and event-driven programming.

    Asynchronous Pattern Playback: Utilized setTimeout loops to handle the timing of the computer's sequence animations.

    State Tracking: Implemented game logic using arrays to compare the gameSequence against the userSequence in real-time.

    Dynamic UI Updates: Leveraged CSS classes and DOM manipulation to provide instant visual feedback (flashing effects) for both the AI and the player.

    Event Handling: Used centralized event listeners to manage user inputs efficiently across multiple game elements.

🛠️ Built With

    HTML5: Semantic structure for the game board.

    CSS3 (Flexbox/Grid): For a responsive, centered layout that works on mobile and desktop.

    Vanilla JavaScript: Pure logic without the use of external libraries, focusing on raw DOM API proficiency.

🎮 How to Play

    Press any key to start the game.

    The game will flash a color. Remember the sequence.

    Click the colors in the exact same order.

    Each successful round adds a new step to the sequence. If you miss one, it's Game Over!
